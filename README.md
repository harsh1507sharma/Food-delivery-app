# Food Delivery App

A full-stack food delivery application built as a learning project using the MERN stack.

## Features
- User authentication
- Browse food items
- Cart management
- Place orders
- Admin panel

## Tech Stack
### Frontend
- React
- Vite
- Context API

### Backend
- Node.js
- Express
- MongoDB

## Environment Variables
Create `.env` files in frontend and backend directories as required.

## How to Run Locally
1. Clone the repository
2. Install dependencies in both frontend and backend
3. Add environment variables
4. Run frontend and backend servers
